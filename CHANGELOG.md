# Changelog

All notable releases of [Murmur](https://murmurtts.com/) are documented here. Latest builds are always available at [murmurtts.com](https://murmurtts.com/).

## [1.0.6] - 2026-04

- Fix Fish and Qwen install failures on Python 3.13+
- Improved Python environment setup reliability on first launch

## [1.0.5] - 2026-04-13

- Sparkle update key rotation for stronger update integrity
- Note: 1.0.4 installs cannot auto-update to 1.0.5 or later. Please [download the latest build manually](https://murmurtts.com/) and reinstall.

## [1.0.4]

- Stability and performance improvements
- Voice library refinements
